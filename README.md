## How to Run the Program

To set up and run the program:

```bash
# Clone the repository
git clone https://github.com/SpicyNachos03/movie-booking-website-csci4050.git

# Install dependencies
npm install

# Run the backend
cd backend
npm restart

# Run the frontend
npm run dev
